Modificar Contenido HTML

En esta sección, exploraremos cómo JavaScript puede cambiar el contenido de elementos HTML en tiempo real. A continuación, te presentamos un ejemplo en el que se actualiza un mensaje al hacer clic en un botón:

Este es un mensaje original.

En el ejemplo anterior, tenemos un párrafo con el texto "Este es un mensaje original". Al hacer clic en el botón, JavaScript cambia el contenido de ese párrafo a "Este es un mensaje actualizado con JavaScript.". Esto ilustra cómo JavaScript puede interactuar con elementos HTML y modificar su contenido en tiempo real.
